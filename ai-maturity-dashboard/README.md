# AI Maturity Tracking Dashboard

A client-only React + TypeScript dashboard for tracking agentic AI maturity across key organizational areas.

## Features

- **Overview Dashboard**: Displays overall maturity score and individual bucket scores
- **Bucket Details**: Detailed view for each maturity area
- **Gauge Indicators**: Visual representation of maturity stages
- **Configurable Data**: Edit `src/data/maturityData.json` to update maturity scores

## Maturity Buckets

1. SDLC Adoption
2. People
3. GRC (Governance, Risk, Compliance)
4. Infrastructure
5. Investment
6. Cross AI Initiatives
7. Management Engagement

## Maturity Stages

- Not Started (0)
- Beginning (1)
- Developing (2)
- Mature (3)
- Advanced (4)

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Configuration

Edit `src/data/maturityData.json` to update maturity scores and descriptions for each bucket.

## Tech Stack

- React 18 with TypeScript
- Vite
- React Router DOM
- CSS Modules
