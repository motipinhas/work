AI Maturity Dashboard - Installation Instructions
==================================================

Thank you for receiving the AI Maturity Dashboard!

QUICK START:
------------

1. Extract this zip file to a folder on your computer

2. Open Terminal (Mac/Linux) or Command Prompt (Windows) in the extracted folder

3. Run the server:
   
   On Mac/Linux:
   ./run-server.sh
   
   On Windows:
   run-server.bat

4. Open your web browser and go to:
   http://localhost:3000

5. The dashboard should now be running!

REQUIREMENTS:
-------------
- Node.js (version 18 or higher) OR Python 3
- A modern web browser (Chrome, Firefox, Safari, or Edge)

TROUBLESHOOTING:
----------------
- If port 3000 is busy, you can use a different port:
  Mac/Linux: ./run-server.sh 8080
  Windows: run-server.bat 8080
  Then access: http://localhost:8080

- If you see "dist directory not found":
  The build files are missing. Please contact the sender.

- To stop the server:
  Press Ctrl+C in the terminal/command prompt
